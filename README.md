# Cultural Geography in Information Environment

Проект выполнен в рамках МДКП «Культурная география в информационной среде».

## Описание проекта

Репозиторий содержит материалы курсового проекта, связанного с обработкой, структурированием и хранением данных о культурно-географических объектах.

В проекте реализованы:

- создание базы данных;
- работа с таблицами и связями между сущностями;
- SQL-запросы для получения данных;
- CRUD-операции на Python;
- обработка данных в Jupyter Notebook.
